  • ออกแบบ UX/UI สำหรับเว็บไซต์เกมเพื่อการเรียนรู้ภาษาจีนระดับ HSK 1–3 ภายใต้โปรเจกต์ “PANDA WORDS”

  • ออกแบบ Responsive Layout เพื่อให้เกมสามารถใช้งานได้ดี ทั้งบน Desktop, Tablet และ Mobile
Front-end Development

2025 - 2026

PANDA WORDS – Chinese Learning Game Website (HSK 1-3)

  • พัฒนาเว็บไซต์เกมเพื่อการเรียนรู้ภาษาจีนระดับ HSK 1–3 โดยรับผิดชอบงานด้าน Front-end Development
  • พัฒนาเว็บไซต์ด้วย Next.js โดยใช้ JavaScript และ CSS ผ่าน Visual Studio Code พร้อมออกแบบระบบแสดงผลของเกม (Rendering Logic)
  • ออกแบบและพัฒนามินิเกมเพื่อการเรียนรู้จำนวน 4 รูปแบบ ได้แก่ เกมเติมประโยคให้ถูกต้อง, เกมเรียงประโยคให้ถูกต้อง, เกมฟังเสียงแล้วเลือกคำตอบที่ถูกต้อง และเกมสะกดคำภาษาจีนให้ถูกต้อง
  • พัฒนา Game Logic ของแต่ละเกมให้มีระบบตรวจคำตอบ และการคำนวณคะแนนที่แตกต่างกันตามประเภทของเกม
  • เชื่อมต่อระบบกับฐานข้อมูล Firebase เพื่อจัดการข้อมูลผู้เล่น และปรับเว็บไซต์ให้รองรับการใช้งานทุกอุปกรณ์ (Responsive Design)
  • ทดสอบการใช้งาน (User Testing) และปรับปรุง UX ตาม Feedback เพื่อเพิ่มประสิทธิภาพในการเรียนรู้และความสนุกของผู้เล่น
IT Support

2025-2026

IT Support Training

  • อัปเดตและติดตั้งระบบคอมพิวเตอร์ภายในหน่วยงาน พร้อมตรวจสอบความพร้อมของระบบให้สามารถใช้งานได้อย่างมีประสิทธิภาพ
  • แก้ไขปัญหาเบื้องต้นของเครื่องคอมพิวเตอร์ เช่น เครื่องทำงานช้า รวมถึงตรวจสอบและปรับปรุงประสิทธิภาพของระบบ
  • ตรวจสอบ ดูแล และจัดการซอฟต์แวร์ภายในเครื่อง พร้อมแก้ไขปัญหาการใช้งานเบื้องต้นของผู้ใช้
  • ตรวจสอบและแก้ไขปัญหาอุปกรณ์สำนักงาน เช่น เครื่องพิมพ์และโปรเจคเตอร์ รวมถึงการเชื่อมต่อระหว่างอุปกรณ์กับคอมพิวเตอร์
  • แก้ไขปัญหาการแสดงผลของโปรเจคเตอร์และการเชื่อมต่อผ่านแอปพลิเคชัน เพื่อให้สามารถใช้งานได้เต็มประสิทธิภาพ
  • ตรวจสอบและแก้ไขปัญหาเครื่องพิมพ์ที่ไม่สามารถพิมพ์งานได้ รวมถึงการตั้งค่าการเชื่อมต่อกับคอมพิวเตอร์
  • ตรวจสอบและจัดการสต็อกอุปกรณ์ IT โดยแยกประเภทอุปกรณ์ที่สามารถใช้งานได้และไม่ได้ พร้อมจัดเก็บอย่างเป็นระเบียบ
  • ติดตั้งและตั้งค่าคอมพิวเตอร์เครื่องใหม่ให้พร้อมใช้งานภายในองค์กร
  • จัดทำเอกสารและบันทึกวาระการประชุม เพื่อสนับสนุนการทำงานภายในสำนักงาน
  • ออกแบบโปสเตอร์ด้านการแพทย์แผนจีน (ฝังเข็ม) ให้มีความสวยงาม อ่านง่าย และเหมาะสำหรับใช้เป็นสื่อเผยแพร่ข้อมูล
